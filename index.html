Project 2

Tree Adoption App (SDG 15)

A mobile application developed using Jetpack Compose, Firebase and Room Database. The application supports SDG 15 (Life on Land) by encouraging users to adopt trees and learn about environmental conservation.

Features:

  • Home Dashboard
  • Tree Adoption Form
  • Park Trees Gallery
  • Ancient Trees Gallery
  • History Tracking
  • Firebase Cloud Storage
  • Location Service
